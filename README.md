# Teslo API
1. clonar el proyecto
2. copiar el archivo __.env.template__ y renombrarlo a __.env__
3. cambiar las variables de entorno
4. levantar la base de datos con docker
```
docker-compose up -d
```
6. levantar:
```
 yarn start:dev
```
